Wat ik voor je kan doen

Uitgebreide dienstverlening rondom Microsoft 365, Azure en maatwerksoftware. Elke opdracht is maatwerk — van eenmalig project tot structureel beheer.

Security & Zero Trust

Een veilige Microsoft 365-omgeving begint bij een degelijke identiteitsstrategie. Ik configureer Conditional Access policies, rol MFA uit voor alle gebruikers en zet Identity Protection in voor risicodetectie. Entra ID vormt het fundament van het Zero Trust-model: geen vertrouwen op basis van locatie, maar op basis van identiteit en apparaatstatus. Aangevuld met Microsoft Defender en Purview voor databescherming en compliance.

Entra ID Defender Purview Conditional Access MFA
Device Management

Moderne werkplekbeheer via Microsoft Intune. Ik stel compliance-beleid in, configureer Autopilot v2 voor zero-touch deployment en implementeer Windows Defender Application Control (WDAC) voor maximale beveiliging. Nieuwe medewerkers hoeven hun laptop alleen maar op te starten — de rest gaat automatisch. Inclusief app-deployments, updates en rapportage via het Intune-portal.

Intune Autopilot WDAC Compliance Windows
Automatisering

Repetitief beheerwerk hoort geautomatiseerd te zijn. Ik schrijf PowerShell-scripts voor alles van user provisioning en licentietoewijzing tot security-rapportage en tenant-analyse. Microsoft Graph API maakt het mogelijk om volledig programmatisch met M365 te werken. Infrastructure as Code via Bicep zorgt ervoor dat Azure-omgevingen herhaalbaar en gedocumenteerd worden ingericht.

PowerShell Graph API Bicep Automatisering
Azure & Cloud

Azure-infrastructuur opgezet als Infrastructure as Code. Governance-beleid, resource management en kostenoptimalisatie zorgen voor een beheersbare cloudstrategie. Ik werk met Azure Bicep en ARM-templates voor reproduceerbare deployments. Van resource groups en policies tot monitoring met Azure Monitor en Log Analytics — alles gedocumenteerd en gestructureerd.

Azure Bicep ARM Governance IaC
Portals & Dashboards

Op maat gebouwde webapplicaties die Microsoft 365-data inzichtelijk maken. Via Microsoft Graph API haal ik real-time gegevens op over gebruikers, licenties, security en apparaten. De frontend is gebouwd in React, de backend in FastAPI (Python). Klanten krijgen een eigen portaal waarop ze zelf de gezondheid van hun tenant kunnen monitoren — zonder technische kennis.

Migraties & Onboarding

Gestructureerde tenant migraties van A tot Z. Ik start met een uitgebreide tenant-intake: inventarisatie van gebruikers, licenties, security-instellingen en verbeterpunten. Op basis hiervan volgt een helder migratieplan. Nieuwe klanten worden gedocumenteerd ingewerkt — standaard werkwijzen, runbooks en configuratieoverzichten zorgen voor continuïteit en controle.
